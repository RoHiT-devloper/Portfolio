### Updating Resume PDF
1. Upload your resume PDF to a cloud storage service (Google Drive, Dropbox, etc.)
2. Update the `resumePDF` link in the `CONFIG` object in `script.js`:

```javascript
const CONFIG = {
    resumePDF: "https://your-new-resume-link.com/rohit-dutta-resume.pdf",
    // ... other config
}; -->
