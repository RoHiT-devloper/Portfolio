## ğŸ“¦ Installation & Setup

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)

### Quick Start
1. **Clone the repository**
   ```bash
   git clone https://github.com/RoHiT-devloper/3d-portfolio.git
   cd 3d-portfolio
   ```

2. **Open the project**
   - Simply open `index.html` in your browser, or
   - Use a local server for best performance:
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js
   npx serve
   ```

3. **Customize for your use**
   - Update personal information in the HTML
   - Replace profile images in the `./` directory
   - Modify color schemes in CSS variables
   - Add your own projects and certificates

## ğŸ¯ Customization Guide

### Personal Information
Edit the following sections in `index.html`:
- Personal details in Hero section
- Professional experience
- Projects and descriptions
- Education background
- Contact information

### Styling & Themes
Modify CSS variables in `:root` and `.dark-theme`:
```css
:root {
    --primary: #2563eb;      /* Main brand color */
    --secondary: #10b981;    /* Accent color */
    --bg-color: #f9fafb;     /* Background color */
    /* ... other variables */
}
```

### Adding New Projects
Update the `projects` object in JavaScript:
```javascript
const projects = {
    yourProject: {
        title: "Project Name",
        description: "Project description...",
        technologies: ["Tech1", "Tech2", "Tech3"],
        features: ["Feature 1", "Feature 2"],
        liveLink: "https://your-project.com",
        githubLink: "https://github.com/your-repo"
    }
};
```
